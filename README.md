# CHANITHU MD 👑
🤖 **WhatsApp Automation Mini Bot**

## 🟢 Status
* **Uptime:** 24/7 Online
* **Hosting:** Linux OS (Cloud Hosted)
* **Prefix:** `!`

## 🛠️ Main Features
* **Auto Reply:** Responds instantly to greetings like `hi`.
* **Menu System:** Built-in rich menu text using `!menu`.
* **Multi-Device Support:** Stays online even when your phone is disconnected.
